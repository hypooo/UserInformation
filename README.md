### 简单的用户信息管理系统
#### 技术总结
- Jdk 1.8
- Mysql 5.7.26
- jsp & servlet
- Apache Tomcat 9.0.21
- Apache BeanUtils
- Druid connection pools
- Spring Jdbc Template

#### 功能
- 管理员登录
- 用户信息的添加
- 用户信息的删除
- 用户信息的修改
- 用户信息的整体查询
- 用户信息的分页条件查询
